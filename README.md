# Vimouse  
*A keyboard-driven mouse controller for Vim lovers*

## Overview

**Vimouse** is a lightweight utility designed for Vim enthusiasts who want to control the mouse entirely from the keyboard‚Äîno hand movement required. Navigate, click, scroll, and tag positions on screen using intuitive, Vim-inspired keybindings.

Stay in the flow. Keep your fingers on the home row.

---

## Keybindings

### Entering Normal Mode
- **Activate Normal Mode**: `Ctrl + J` or `Ctrl + Alt + K`  
  - Using `Ctrl + Alt + K` centers the cursor on screen automatically.

### Exiting Any Mode 
- **Exit mode**: `Esc` or `Enter`  
  - Pressing `Enter` exits and **automatically left-clicks**, useful for focusing the current window.

---

### Normal Mode
- `h` / `j` / `k` / `l` ‚Üí Move mouse left / down / up / right  
- `u` / `o` / `n` / `.` ‚Üí Move diagonally: ‚ÜñÔ∏è / ‚ÜóÔ∏è / ‚ÜôÔ∏è / ‚ÜòÔ∏è  
- `f` ‚Üí Left mouse click  
- `g` ‚Üí Right mouse click  
- `r` ‚Üí Return to the **previous** clicked position  
- `e` ‚Üí Go to the **next** clicked position (in history)  
- `y` ‚Üí Enter **Wheel Mode**  
  - In Wheel Mode, use `h`/`j`/`k`/`l` to scroll horizontally or vertically (simulates mouse wheel)  
- `i` ‚Üí Enter **Grid Mode**  
- `m` ‚Üí Enter **Hint Mode** (also called "Map Mode")  
- `q` ‚Üí Toggle **Tag** at current mouse position  
  - Creates a single-letter tag (e.g., `A`, `B`) at the cursor location  
  - If a tag already exists there, it is removed  
- `w` ‚Üí Enter **Tag Mode**

---

### Tag Mode
- All defined tags appear highlighted on screen  
- Press the corresponding letter (e.g., `A`) to **move the cursor to that tag**  
- Tag Mode exits automatically after navigation

---

### Grid Mode
- `Q` / `W` / `A` / `S` ‚Üí Jump mouse to one of the four screen quadrants:  
  - `Q`: Top-left‚ÄÉ‚ÄÉ`W`: Top-right  
  - `A`: Bottom-left‚ÄÉ`S`: Bottom-right  
- Pressing `h`/`j`/`k`/`l` in Grid Mode **exits back to Normal Mode**

---

### Hint Mode (Map Mode)
- After pressing `m`, the screen is overlaid with two-letter coordinate hints (e.g., `AA`, `AB`, `AC`, ‚Ä¶)  
- Type any **two-letter combination** (e.g., `AB`) to instantly move the mouse to that region  
- `Esc` ‚Üí Exit without clicking  
- `Enter` ‚Üí Exit and **left-click** (useful for activating UI elements)  
- `r` ‚Üí Return to the last grid/hint block used

---

## Additional Features
- ‚úÖ **Non-intrusive**: Common system shortcuts (e.g., `Ctrl+C`, `Ctrl+V`, `Ctrl+A`) are **passed through unaffected**  
- ‚úÖ **Zero dependencies**: Runs as a single executable  
- ‚úÖ **Instant launch**: No setup or configuration needed  
- ‚úÖ **Low resource usage**: Minimal CPU and memory footprint

---

## Inspiration
Vimouse is heavily inspired by [**warpd**](https://github.com/rvaiya/warpd), but reimagined with a keybinding layout optimized for personal workflow and Vim muscle memory.
